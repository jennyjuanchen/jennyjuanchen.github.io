High-performance computing: Study the parallel algorithms on multi-core processor systems; Optimize the scalable linear algebra libraries for parallel and distribued computing systems; Study the load balance on CPU/GPU heterogeneous systems; Study the parallel task assignment on many-core processor systems and so on.

Energy-Efficiency Optimization: How to improve energy efficiency on CPU, GPU, CPU/GPU systems while maintaining a certain performance? Howadays, DVFS techniques have been widely used for reducing power/energy consumption. Besides DVFS, undervolting, overclocking techniques are also used to boost the energy efficiency.

Power Management Techniques for Supercomputer Systems: How to manage power consumption in a supercomputer center is a significant problem, which has been concerned by computer system designers and software developers. For a supercomputer center, compute node, interconnection network, storage system, and so on need to be managed to obtain lower power consumption.

Parallel Software Development Framework: Many scientific computing applications need to be developed aided by software framework, which can reduce the degree of software development difficulty and makes it easy to maintain the applications in the future. For example, CFD simulation needs computer scientists and physicists to cooperate together to achive a higher simulation level.

Computer Theory: Study network topology partition methods during HPC interconnection networks for improving the energy manamgement effectiveness; Study the thermal model and fault tolerance model on multi-core processor systems in order to furthermore improve the energy efficiency by DVFS, undervolting or overclocking techniques; Study the scalability model for the next generation supercomputer systems, and so on. All the theory problems come from the real situations and all the theorial results will hope to be validated on the real platform, which make our computer theory reseaches more significant.
