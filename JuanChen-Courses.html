Graduate Courses
The first-year graduate students at NUDT
The first-year graduate students at Hunan University

Objective: teach graduates parallel computer architecture, parallel algorithms, and parallel programming methodology on the Tianhe-2 and Tianhe-1A

Textbooks: Introduction to Parallel Computing (2nd Edition), by Ananth Grama, George Karypis, Vipin Kumar, Anshul Gupta.
Reference Books: An Introduction to Parallel Programming, by Peter Pacheco.

Contents:
* parallel computer architecture
* shared-memory and distributed-memory programming, such as MPI/OpenMP/Pthreads
* parallel programming model
* basic parallel algorithms
* performance evaluation, debugging
* advanced parallel optimization techniques
* real applications optimization

Projects: Every student needs to take part in at least one of the projects below:
* Parallel Optimization of Spectrum Transformation Shallow Water Wave Mode
* Load Balance Optimization for CFD-MD Coupling Computing
* Parallel Mining of Biomedical Literature
* Cell Response Big Data Based Biological Effect Evaluation
* 3D Simulations of Human Ventricular Tissues
* DPM based Radiotherapy Parallelization
* PETSC based CG Parallel Tolerance Optimization
